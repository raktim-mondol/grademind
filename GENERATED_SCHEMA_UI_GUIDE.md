## When Does It Appear?

The Generated Schema section only appears when:

1. âœ… Assignment processing is complete (`processingStatus === 'completed'`)
2. âœ… Rubric has been processed (`rubricProcessingStatus === 'completed'`)
3. âœ… Schema extraction succeeded (`gradingSchemaStatus === 'completed'`)
4. âœ… `gradingSchema` field is not null/empty

### If Schema is Not Available

If the schema hasn't been extracted yet or extraction failed, the section will not appear in the modal. The other three sections (Assignment Structure, Rubric Criteria, Solution Key) will still be visible if their data is available.

## Styling Details

### Colors
- **Active accordion**: `bg-zinc-900` (black) with `text-white`
- **Inactive accordion**: `bg-zinc-50` (light gray) with `text-zinc-600`
- **Hover state**: `bg-zinc-100` (slightly darker gray)
- **Border**: `border-zinc-200` (2px solid)
- **JSON background**: `bg-zinc-50` with `border-zinc-100`

### Typography
- **Header**: Uppercase, bold, small font (`text-xs font-bold uppercase tracking-wider`)
- **JSON content**: Monospace font (`font-mono text-xs`)
- **Color**: `text-zinc-700` for JSON content

### Spacing
- **Padding**: `px-4 py-3` for header, `p-4` for content
- **Border**: `border-2` for outer border, `border-t-2` for divider
- **Max height**: `max-h-96` (384px) with scrolling

## Responsive Design

The modal and accordion sections are fully responsive:
- **Mobile**: Full-width modal with stacked accordions
- **Tablet**: Same layout with adjusted padding
- **Desktop**: Centered modal with optimal width

## Keyboard Accessibility

- **Tab**: Navigate between accordion headers
- **Enter/Space**: Toggle accordion expand/collapse
- **Escape**: Close the modal (if implemented)
